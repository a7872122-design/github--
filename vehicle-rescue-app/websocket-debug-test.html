📊 診斷信息

測試步驟：

  1. 點擊「連接為廠商」以啟動廠商連接
  2. 檢查廠商控制台是否顯示「已連接」日誌
  3. 點擊「連接為用戶」以啟動用戶連接
  4. 點擊「發送救援請求」即時發送救援請求
  5. 檢查廠商控制台是否收到救援請求消息
  6. 打開瀏覽器開發工具（F12）查看後端伺服器日誌

故障排查：

  • 如果廠商未收到消息，檢查廠商 ID 是否與發送者的目標 ID 相同
  • 檢查 WebSocket 連接狀態是否為「已連接」
  • 查看瀏覽器控制台 (F12)，查看是否有連接錯誤
  • 檢查後端伺服器日誌是否顯示客戶端註冊和廣播嘗試
